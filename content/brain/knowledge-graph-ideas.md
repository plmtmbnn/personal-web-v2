The Knowledge Graph is a visual rendering of the notes in this vault.

### Core Concepts:
- **Nodes**: Represent individual notes (like this one).
- **Edges (Links)**: Represent connections between notes, created using the `[[WikiLink]]` syntax.
- **Backlinks**: Allow you to see which notes reference the current note.

This visual representation makes it easy to spot clusters of thoughts and navigate from one topic to another. It uses D3's force-directed simulation via the browser canvas API for high performance.
For more details, head back to [[Welcome to My Second Brain]].
